# 🍳 Kitchen Bot — Telegram Web App для кухни

Система управления кухней ресторана через Telegram Mini App.

## Возможности

### Роли
| Функция | Повар | Су-шеф/Шеф |
|---------|:-----:|:----------:|
| Открыть/закрыть смену | ✅ | ✅ |
| Просмотр ревизии | ✅ | ✅ |
| Ввод фактического количества | ✅ | ✅ |
| Добавить/удалить ингредиент | ❌ | ✅ |
| Установить минимальный остаток | ❌ | ✅ |
| Просмотр ТТК | ✅ | ✅ |
| Создать/редактировать/удалить ТТК | ❌ | ✅ |
| Список «К заказу» | ❌ | ✅ |

### Вкладки
- **Смена** — открытие/закрытие смены с логированием времени
- **Ревизия** — учёт ингредиентов, автоматическое добавление в «К заказу» при низком остатке
- **ТТК** — технико-технологические карты блюд
- **К заказу** — список ингредиентов для закупки (только шефы)
- **Роли** — управление ролями пользователей (только админ)

## Пользователи и роли

**Добавлять людей вручную не нужно.**

1. Отправьте сотруднику ссылку на бота (`t.me/имя_вашего_бота`)
2. Он нажимает `/start` → «Открыть кухню»
3. Человек автоматически регистрируется как **Повар**
4. Вы (админ) открываете приложение → вкладка **«Роли»** → меняете роль на **Су-шеф/Шеф**

Админ — тот, чей Telegram ID указан в `ADMIN_TELEGRAM_ID`. Узнать ID: `/myid` в боте.

Подробный деплой: **[DEPLOY.md](DEPLOY.md)**

## Быстрый старт

### 1. Создайте бота в Telegram

1. Напишите [@BotFather](https://t.me/BotFather)
2. `/newbot` → задайте имя и username
3. Скопируйте токен

### 2. Узнайте свой Telegram ID

Напишите [@userinfobot](https://t.me/userinfobot) — он пришлёт ваш ID.

### 3. Настройте окружение

```bash
cp .env.example .env
```

Заполните `.env` (не путайте токен и ID!):
```
BOT_TOKEN=123456789:ABC...     # от @BotFather
ADMIN_TELEGRAM_ID=729409944    # число от /myid или @userinfobot
WEBAPP_URL=https://ваш-url.com
WEBHOOK_MODE=false             # true только на сервере
API_PORT=8000
```

### 4. Установите зависимости

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Linux/Mac
source venv/bin/activate

pip install -r requirements.txt
```

### 5. Запуск

```bash
# API + бот вместе
python run.py

# Только API (для разработки)
python run.py api

# Только бот
python run.py bot
```

### 6. Настройте Web App в BotFather

```
/setmenubutton
→ выберите бота
→ укажите URL: https://ваш-домен.com
→ текст кнопки: Открыть кухню
```

## Локальная разработка

Для тестирования Web App нужен HTTPS. Используйте [ngrok](https://ngrok.com/):

```bash
# Терминал 1
python run.py api

# Терминал 2
ngrok http 8000
```

Скопируйте HTTPS URL из ngrok в `WEBAPP_URL` в `.env`, перезапустите бота.
